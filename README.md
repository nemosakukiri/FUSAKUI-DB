# ⚖️ The Forensic Editorial
### 公務不作為フォレンジック・データベース

「個人の怒りを、社会を変える法的証拠に。」

**The Forensic Editorial** は、役所の窓口で起きる「水際作戦」や「不当な申請拒否」などの行政の不作為を、憲法および行政手続法、福祉関連法規の視点から可視化・構造化する調査報道支援プラットフォームです。

---

## 🚀 プロジェクトのビジョン
窓口での「たらい回し」や「説明拒否」は、単なるマナーの問題ではなく、憲法第25条（生存権）や行政手続法第7条（申請の受理義務）に抵触する可能性がある深刻な問題です。
本プロジェクトは、市民の証言をAI（リーガル・エンジン）によって法的に分析し、統計データとして集積することで、制度改善のための客観的な根拠を提示します。

## 🛡️ 主な機能
- **AIリーガル・スクリーニング**: 投稿内容を憲法・行政法・生活保護法等の視点から即座に分析。違法性の疑い（％）と該当条文を提示。
- **フォレンジック・インタビュー**: 法的証拠としての信憑性を高めるための、構造化されたヒアリング・プロセス。
- **不作為マッピング（順次搭載予定）**: サービス不履行が集中している自治体や窓口をリアルタイムで可視化。
- **証拠のセキュア保管**: 録音データや書類の控えを、プライバシーに配慮した形で暗号化保存。

## 📖 判定基準とする主な法規・規範
- **日本国憲法 第25条**: 全て国民は、健康で文化的な最低限度の生活を営む権利を有する。
- **行政手続法 第7条**: 申請に対する審査、受理義務の遵守。
- **生活保護法 第7条**: 申請保護の原則。
- **厚生労働省通知・判例**: 水際作戦の違法性を指摘する過去の重要な指針。
